Introduction

CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple webpages all at once. It is used to define styles for webpages, including the design, layout and variations in display for different devices and screen sizes.

Some CSS properties used for designing tables on a web page include:

  • Borders
  • Width and Height
  • Alignment
  • Padding
  • Horizontal Dividers
  • Color
  • Responsive Table

Simple Table

The HTML <table> element allows web authors to display tabular data (such as text, images, links, other tables, etc) in a two dimensional table with rows and columns of cells. Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

Example:

                    
                        <table>
                            <tr>
                                <th>Heading 1/Column 1</th>
                                <th>Heading 2/column 2</th>
                            </tr>
                            <tr>
                                <td>Row 1 Data Column 1</td>
                                <td>Row 1 Data Column 2</td>
                            </tr>
                            <tr>
                                <td>Row 2 Column 1</td>
                                <td>Row 2 Column 2</td>
                            </tr>
                        </table>

This will render a <table> consisting of three total rows (<tr>): one row of header cells (<th>) and two rows of content cells (<td>). <th> elements are tabular headers and <td> elements are tabular data. You can put whatever you want inside a <td> or <th>.

Borders

If you do not specify a border for the table, it will be displayed without borders. A border is set using the CSS border property.

The table below specifies a black border for <table>, <th> and <td>:

Firstname Lastname
Peter Griffin
John Doe
Andrew Matthew

Example:

                    
                        table, th, td {
                            border: 1px solid black;
                        }

Notice that the table in the example above has double borders. This is because both the table and the <th> and <td> elements have seperate borders.

The border-collapse property sets whether the table borders should be collapsed into a single border:

Firstname Lastname
Peter Griffin
John Doe
Andrew Matthew

Example:

                    
                        table {
                            border-collapse: collapse;
                        }

                        table, th, td {
                            border: 1px solid black;
                        }

Width

The width and height of a table are defined by the width and height properties. The example below sets the width of the table to 100%, and the height of the <td> elements to 50px:

Firstname Lastname Savings
Peter Griffin $100
John Doe $150
Andrew Matthew $300

Example:

                    
                        table {
                            width: 100%;
                        }
                        th {
                            height: 50px;
                        }

Alignment

The text-align property sets the horizonyal alignment (like left, right, or center) of the content in <th> or <td>. By default, the content of <th> elements are center-aligned and the contents of table cells elements are left-aligned.

The following example left-aligns the text in <th> elements:

Firstname Lastname Savings
Peter Griffin $100
John Doe $150
Andrew Matthew $300

Example:

                    
                        th {
                            text-align: left;
                        }

The vertical-align property sets the vertical alignment (like top, bottom, or middle) of the content in table head or table cells. By default, the vertical alignment of the content in a table is middle (for both ,<th> and <td> elements).

The following example sets the vertical text alignment to bottom for tabular data elements:

Firstname Lastname Savings
Peter Griffin $100
John Doe $150
Andrew Matthew $300

Example:

                    
                        td {
                            height: 50px;
                            vertical-align: bottom;
                        }

Padding

To control the space between the border and the content in a table, use the padding property on <td> and <th> elements:

Firstname Lastname Savings
Peter Griffin $100
John Doe $150
Andrew Matthew $300

Example:

                    
                        td, th {
                            padding: 15px;
                            text-align: left;
                        }
